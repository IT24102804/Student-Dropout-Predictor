# Student Dropout Early Warning System

## Objective
A machine learning classification project designed to identify at-risk students and predict academic dropout rates using demographic and academic data. 

## Dataset
This project uses the "Predict Students' Dropout and Academic Success" dataset from the UCI Machine Learning Repository. It addresses significant class imbalances to provide actionable insights for academic intervention.

## Tech Stack
* **Language:** Python
* **Libraries:** Scikit-learn, Pandas, Seaborn, Matplotlib
* **Environment:** Google Colab

## Results
* **Accuracy:** 88% overall accuracy.
* **Recall:** Successfully identified 82% of actual dropout cases, allowing for high-confidence early intervention.
* **Insights:** Evaluated feature importance to determine the strongest predictors of student attrition.
